# 冰洲石紫外激发红光余晖模拟器

🔮 **Iceland Spar Red Afterglow Simulator**

基于真实实验数据的交互式网页模拟器，用于探究不同温度和紫外照射时间下冰洲石（CaCO₃）的红光余晖持续时间。

## 实验背景

冰洲石（Iceland Spar）是一种高纯度的方解石变种，在紫外光激发下，晶体中的 Mn²⁺ 杂质离子会发出特征性的橙红色余晖（~590nm）。余晖持续时间受温度、激发波长和照射时间等因素影响。

本项目基于视频分析实验数据，建立了双变量数学模型：

```
τ(T, t) = τ₀ · t^α · exp(β·(T - T₀))
```

其中：
- τ₀ = 90.8 ms（室温基准余晖时间）
- α = 0.136（照射时间幂指数）
- β = −0.0445 K⁻¹（温度系数，等效活化能 ≈ 0.34 eV）

## 功能

- 🌡️ 温度控制（0°C – 70°C）
- ⏱️ 紫外照射时间控制（1s – 30s）
- 💜 激发波长选择（313nm – 436nm）
- 📉 实时余晖衰减曲线
- 📊 数据拟合可视化
- 🧬 物理模型能级图

## 技术栈

- 纯 HTML/CSS/JavaScript（Canvas 2D）
- 响应式设计，支持移动端
- 无需任何依赖或构建步骤
