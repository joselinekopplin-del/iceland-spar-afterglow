# 光迹——多维光学现象交互研究平台

**LumiTrace — Interactive Optical Research Platform**

一个可以直接运行在 GitHub Pages 上的交互式光学研究网页。页面采用接近 iOS 的分层玻璃界面、轻量动效和自适应布局，不依赖服务器或第三方网络资源。

在线访问：<https://joselinekopplin-del.github.io/iceland-spar-afterglow/>

## 研究课题

### 课题一：冰洲石红光余晖

- 分析不同温度和紫光照射时间下，切断紫光后的红光余晖持续时间
- 提供实验网格插值与交叉验证温度模型
- 展示预测区间、参数敏感度、实时分析和衰减动画
- 页面内列出完整计算公式

### 课题二：旋光散射

- 与冰洲石余晖数据相互独立
- 计算有效衰减系数与旋光速率
- 拟合散射参数 B、V、β
- 模拟散射光强随传播长度的变化

### 课题三：RGB 主波长与光谱分析

- sRGB、线性 RGB、CIE XYZ 与 xy 色度坐标转换
- 主波长、互补波长、激发纯度和可信度分析
- RGB 占比高斯、灵敏度高斯、光源经验及 PCA 光谱拟合
- 图片点击选点、颜色记录与 PCA 校准模型导入导出

## 使用方式

直接打开根目录的 `index.html`，或通过上方 GitHub Pages 地址访问。所有计算、图片和 CSV 数据均在浏览器本地处理。
