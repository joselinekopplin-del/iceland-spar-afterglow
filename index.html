Physical mechanism

一束紫外光，如何变成红色余晖

从电子被激发、俘获到热释放与复合发光，六个阶段串起完整的微观过程。

①

紫外激发

当紫外光（313–436 nm）照射冰洲石晶体时，光子能量（3.9–2.8 eV）大于 CaCO₃ 的带隙，将价带电子激发到导带，形成自由电子-空穴对。

hν (UV) → e⁻ (CB) + h⁺ (VB)
②

陷阱俘获

导带中的部分自由电子被晶体中的浅陷阱能级（深度~0.34 eV，位于导带下方）俘获，形成亚稳态存储。照射时间越长，被填充的陷阱越多。

e⁻ (CB) → e⁻ (trap)  ·  N(t) = Nmax(1 − e−t/τfill)
③

切断紫光 · 热释放

切断紫光瞬间，陷阱中的电子通过热激活逐渐释放回导带。温度越高，热释放速率越快——这是温度影响余晖时长的核心机制。

k(T) = k₀ · exp(−Ea / kBT)  ·  Ea ≈ 0.34 eV
④

Mn²⁺ 复合 · 红光发射

释放到导带的电子迁移至 Mn²⁺ 发光中心，与先前被俘获的空穴复合并释放能量。这部分能量传递给 Mn²⁺ 离子的 3d 电子，激发其 4G → 6S 跃迁，发射特征橙红光（~590 nm）。

e⁻ (CB) + Mn³⁺ → Mn²⁺* → Mn²⁺ + hν (590nm, 红光)
⑤

余晖衰减

随着陷阱逐渐排空，复合事件减少，红光强度随时间衰减。衰减规律遵循拉伸指数（Kohlrausch 函数），这是无序介质中弛豫过程的典型特征，反映陷阱深度的分布。

I(t) = I₀ · exp(−(t/τ)β)  ·  β ≈ 0.65 (拉伸因子)
⑥

温度猝灭效应

在高温（>60°C）下，除热释放加速外，Mn²⁺ 激发态还可能通过无辐射跃迁（热猝灭）回到基态，进一步削弱红光发射。这导致高温下余晖急剧缩短。

η(T) = [1 + C·exp(−Eq/kBT)]−1  (热猝灭效率)
Sensitivity

哪些因素决定余晖时长

#1 温度 (最关键因素) 高影响

温度通过 Arrhenius 方程控制陷阱热释放速率。每降温 10°C，余晖约延长 1.5–1.6×；从 25°C 降温到 5°C，余晖延长 ~3×。

5°C → ~281ms → 60°C → ~23ms 相差 ~12×
#2 紫外照射时间 中影响

陷阱填充遵循饱和动力学。照射时间从 1s 增至 10s，余晖增加约 40%；超过 30s 后趋于饱和，增量极小。

1s → ~92ms → 30s → ~163ms 相差 ~1.8×
#3 激发波长 中影响

短波长 UV（313nm）比长波长（436nm）光子能量更高，激发效率更高。313nm 激发下余晖约为 436nm 的 1.3–1.5×。

#4 晶体纯度 / Mn²⁺ 浓度 背景因素

Mn²⁺ 是红光发射的激活剂。浓度过低则发光弱，过高则发生浓度猝灭。天然冰洲石中 Mn²⁺ 浓度通常在 10–1000 ppm 范围。

#5 陷阱深度分布 材料特性

陷阱不是单一能级，而是有一定能量分布。浅陷阱（<0.3eV）在室温下快速释放，深陷阱（>1eV）在室温下几乎不参与余晖，仅在热释光（TL）实验中表现为高温发光峰。

Model documentation

完整计算公式

以下公式基于室温实验数据（视频帧分析 + 手动测量）拟合，并结合文献中冰洲石的物理模型构建。

公式 ① — 余晖特征时间 τ(T, t)
τ(T, texp) = τ₀ · texpα · exp[β · (T − T₀)]
τ₀ = 90.84 ms | α = 0.1365 | β = −4.450 × 10⁻² K⁻¹ | T₀ = 298.15 K (25°C)
物理意义：τ₀ 为室温 1s 照射后基准余晖；α 反映陷阱填充的亚线性增长（α < 1 表示趋于饱和）；β 为温度敏感系数，其值由陷阱深度决定。三项相乘完整描述了余晖时长对温度和照射时间的依赖关系。
公式 ② — 余晖衰减曲线（拉伸指数 / Kohlrausch 函数）
I(t) = I₀ · exp[−(t / τ)βstr]
I₀ = 切断时刻的红光初始强度（归一化=1） | βstr = 0.65（拉伸因子） | τ = τ(T, texp) → 公式①
物理意义：βstr = 1 时为纯指数衰减（单一陷阱深度）；βstr < 1 表明存在陷阱深度分布，衰减初期较快（浅陷阱先排空），后期拖尾更长（深陷阱缓慢释放）。βstr = 0.65 是典型的无序介质弛豫特征值。当 t = τ 时，I = I₀/e ≈ 37% I₀。
公式 ③ — 陷阱热释放速率（Arrhenius 定律）
k(T) = k₀ · exp(−Ea / kBT)
Ea ≈ 0.34 eV（等效陷阱深度） | kB = 8.617 × 10⁻⁵ eV/K（Boltzmann 常数） | k₀ ≈ 10⁸ s⁻¹（指前因子，典型值）
物理意义：该公式是温度影响余晖的根本原因。电子从陷阱释放到导带需要克服能量势垒 Ea。温度升高时，晶格振动加剧，电子获得足够热能的概率指数增长。τ ∝ 1/k(T)，因此公式①中的温度项 exp(β·(T−T₀)) 等价于 exp(Ea/kB·(1/T − 1/T₀)) 的一次近似。
公式 ④ — 陷阱填充动力学（照射阶段）
N(texp) = Nmax · [1 − exp(−texp / τfill)]
Nmax = 总陷阱浓度（材料参数） | τfill ≈ 4.0 s（填充时间常数，拟合值）
物理意义：照射时间 texp 越长，被填充的陷阱 N 越多（接近指数饱和）。公式①中的 texpα 项是对该饱和过程的幂律近似。当 texp ≪ τfill 时 N ∝ t（线性增长），当 texp ≫ τfill 时 N → Nmax（饱和）。
公式 ⑤ — 高温热猝灭效率
η(T) = [1 + C · exp(−Eq / kBT)]−1
C ≈ 10⁶（猝灭常数） | Eq ≈ 0.6–0.8 eV（Mn²⁺ 激发态热猝灭活化能）
物理意义：高温下 Mn²⁺ 激发态通过多声子无辐射弛豫回到基态，不发射光子。该效应在 T > 60°C 开始显著，是高温区域余晖额外缩短的原因。η(T) 描述发光量子效率随温度的变化。
